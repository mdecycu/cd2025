作業一:

請每六人一組從 Week 1 開始執行期中"彈珠檯虛實整合系統"專案, 其內容為探討如何結合 Web 機電模擬場景 (CoppeliaSim 與 Webots) 以及 Arduino based 控制系統, 建置一個虛實整合彈珠檯系統.

首先請各組利用 Solvespace 或 NX2312 所建置的彈珠檯零組件, 透過 3D 列印零件, 組合成一台可以實際發球並以左右兩側搖桿推動彈珠的實體彈珠檯. 其中包括 Flipper, Bumper, Kicker, Score Board 等零組件.

Week 3 上課時段, 請各組進行作業一的分組報告.

作業二:

使用 Solvespace 或 NX2312 零組件,建立 Webots Pinball Machine 模擬系統.

從 Webots Tutorial 入門

根據 https://cyberbotics.com/doc/guide/web-server 中的說明, 建立 Web server.
